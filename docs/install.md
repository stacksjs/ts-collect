# Install

## Bun & Node.js

```bash
bun install ts-collect
npm install ts-collect

# or, invoke immediately
bunx ts-collect
npx ts-collect
```

_We are looking to publish this package npm under the name `ts-collect`. We are also hoping npm will release the name for us._
