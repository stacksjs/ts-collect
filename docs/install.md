# Install

Installing `ts-collect` is easy. Simply pull it in via your package manager of choice.

::: code-group

```sh [npm]
npm install ts-collect
# npm i ts-collect
```

```sh [bun]
bun install ts-collect
# bun add ts-collect
```

```sh [pnpm]
pnpm add ts-collect
```

```sh [yarn]
yarn add ts-collect
```

:::
